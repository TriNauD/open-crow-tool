# Crow 划词插件 — 使用手册（简明版）

面向：**第一次用扩展、不熟悉技术的用户**。  
假设：你已在 Crow **网站**上有账号（没有的话请先在网站注册）。

---

## 插件能帮你做什么

- 在**任意网页**里选中一段话，弹出 **AI 解释**小窗口，不用复制到别处。
- 可以把解释或原文**存进笔记本**，和网站上看到的是同一份数据。
- 账号只在 **Crow 网站**登录即可；插件通过「连接」使用同一个登录状态。

---

## 使用前需要

- 电脑安装 **Google Chrome**（或其它官方说明支持的 Chromium 类浏览器；以下按 Chrome 来写）。
- 能正常打开 Crow **官方网站**并登录（网址以团队提供的为准，例如正式环境域名）。

---

## 第一步：安装扩展

按团队给出的方式二选一即可。

### 方式 A：从 Chrome 应用商店安装（若已上架）

1. 打开管理员发你的 **应用商店链接**。
2. 点 **「添加至 Chrome」**，按提示确认。

### 方式 B：使用离线扩展包（内测或未上架时常用）

1. 向管理员要到**打好包的扩展文件夹**（文件夹里应能看到 `manifest.json` 等文件；不要只拿到其中一个文件）。
2. 在 Chrome 地址栏输入 **`chrome://extensions/`** 并回车。
3. 打开右上角的 **「开发者模式」**。
4. 点 **「加载已解压的扩展程序」**。
5. 选中上一步那个**完整文件夹**，确定。
6. 建议点浏览器工具栏里的 **拼图图标**，把 Crow 扩展 **固定**，方便以后点。

安装后若 Chrome 提示风险：侧载扩展是 Chrome 的正常提示；仅安装你**信任的来源**提供的包。

---

## 第二步：把网站账号「接到」插件（必做）

插件**不会**自己猜你的密码，必须从你已经登录的 **Crow 网页**点一次同步。

1. 在 Chrome 里打开 **Crow 网站**（环境要和扩展要用的一致：一般用正式站就始终用正式站）。
2. **登录**你的账号（若未登录）。
3. 在页面**顶部导航**找到 **「连接插件」** 按钮，**点一次**。
4. 若页面提示成功，即表示插件已拿到当前登录状态，可以开始划词。

**之后如果：**

- 你在网站上 **退出登录** / **换了测试站和正式站** / **很久没用**导致划词失败，  
  请再回到 **对应环境的网站** 登录，并 **重新点一次「连接插件」**。

---

## 第三步：日常使用

1. 打开任意文章或网页，**用鼠标选中**你想理解的一段文字。
2. 选中后附近会出现 **橙色小按钮**，点它打开解释卡片。  
   也可以试快捷键：**Windows / Linux：`Alt + W`**；**Mac：`Ctrl + Shift + W`**（以你电脑实际为准，若冲突可在扩展说明里查看）。
3. 在卡片里可以：看流式解释、把内容 **存入笔记本**、**打开笔记本**网页、点空白处或按 **Esc** 关闭。

---

## 常见问题（很快自查）

| 现象 | 可先这样做 |
|------|------------|
| 选中字后没有橙色按钮 | 确认本页是否禁止脚本（极少数页面）；刷新页面后再试；确认扩展已启用。 |
| 提示登录 / 授权 / 连接 | 打开 Crow 网站 → 登录 → 再点 **「连接插件」**。 |
| 提示网络错误、一直加载 | 确认能正常打开 Crow 网站；换网络或稍后再试。 |
| 想确认插件连的是哪个环境 | 点工具栏里的扩展图标，进入 **选项 / 设置** 页，查看是否显示「已连接」及网站地址（具体文案以当前版本为准）。 |
