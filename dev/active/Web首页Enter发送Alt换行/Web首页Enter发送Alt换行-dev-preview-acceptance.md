# Web 首页 Enter 发送 — Vercel Preview（dev）最小验收

**Preview URL**：`_合并 dev 后替换为本次 Deployment 链接_`

**前置**：桌面浏览器；本地亦可 `npm run dev`。

## 步骤

1. 打开首页，输入 `hello` 后按 **Enter** → 出现解释卡片。
2. 清空后再输入 `a`，按 **Alt+Enter**，再输入 `b`，框内应为两行；此时仍无新卡片；再按 **Enter** → 发送，卡片输入内容含换行（两行）。
3. 输入一段文字，光标在中间按 **Shift+Enter** → 插入换行且不发送。
4. （可选）**⌘/Ctrl+Enter** → 仍可发送。
